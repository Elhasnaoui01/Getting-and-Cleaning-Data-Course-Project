## Getting and Cleaning Data - Course Project

This repository contains the R script and documentation files for the Data Science's track course "Getting and Cleaning data", available in coursera.

## Files

- `run_analysis.R`: R script to perform the analysis.
- `CodeBook.md`: Describes the variables, data, and transformations.
- `FinalData.txt`: The output tidy data set.

## Instructions

1. Download the dataset from [this link](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip).
2. Unzip the dataset to your working directory.
3. Run the `run_analysis.R` script.
4. The `FinalData.txt` file will be generated in your working directory.
